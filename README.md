## 2️⃣ AI-Powered FAQ Chatbot

An intelligent FAQ chatbot that matches user queries with predefined FAQs using NLP similarity techniques.

### 🔹 Features
- Interactive chat interface
- Sidebar FAQ categories
- Quick question buttons
- Cosine similarity-based matching
- Dark mode toggle
- Clear chat option
- Live deployment

### 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity
- Streamlit
